# Проект: Место

Интерактивная страница где можно редактировать свой профиль,добавлять картинки и ставить лайки.

### :hammer_and_wrench: Технологии :
* Flexbox
* Grid Layout
* Media queries
* Файловая структура БЭМ (Nested)
* Сборка проекта с помощью Webpack
